# Michael Yurachek | Socials & Projects

This is the official Socials and Projects page for Michael Yurachek. It displays all social media links and a few personal projects in a clean, minimal interface.

The site is built with **React** and **Vite**, and is hosted on the subdomain [socials.michaelyurachek.com](https://socials.michaelyurachek.com).

## Features

- List of social media links
- Showcase of personal projects
- Responsive design for mobile and desktop
- SEO-friendly with `robots.txt` and `sitemap.xml`
- 404 page for unmatched routes

## Tech Stack

- React 19  
- Vite 8  
- React Router DOM 7  
- CSS / Vanilla styling

## Installation

Clone the repo:

```bash
git clone https://github.com/yourusername/socials.git
cd socials
npm install
npm run dev
```
